    1. What is Semantic HTML? 
    Semantic HTML is used to give meaning to HTML text. It makes it easier for other developers to make sense of what you've written.

    2. What is HTML used for? 
    HTML is used as the first building block of a website, the basic skeleton of what eventually will be a full fledged website.

    3. What is an attribute and where do we put it? 
    Attributes are used in CSS to target specific pieces of text that contain a specified 'attribute' e.g. [title~=MyHeader]
    4. What is the h1 tag used for? How many times should I use it on a page?
    h1 tag is used to denote something as a header. It should be used only once.

    As many times as you want, but ID's should be used only once, for semantic reasons.
